구현한 내용

1. 화면의 좌측, 우측 끝에서 반대방향으로 도형이 사선으로 날라갑니다
2. 화면을 넘어 가면 도형이 사라집니다.
3. 도형은 오각형과 사각형이 날라옵니다.
4. 오각형을 슬라이스 하면 삼각형 하나가 쪼개져 밑으로 떨어지며 남은 사각형이 가던 방향으로 날라갑니다.
5. 사각형을 슬라이스 하면 삼각형 2개로 쪼개지며 밑으로 떨어집니다.
6. 삼각형을 슬라이스 하면 사라집니다.
7. 밑으로 떨어질때 중력가속도가 붙습니다.
8. 밑으로 떨어지는 도형들은 바구니에 닿으면 떨어지는게 멈추며 바구니를 따라 같이 이동합니다.

키보드 명령어

1. w 도형의 모드: LINE/FILL 
2. e 경로출력하기: on/off
3. + 날라오는 속도 0.001 증가 (최대 속도 0.01)
4. - 날라오는 속도 0.001 감소 (최소 속도 0.001)
5. q 프로그램 종료

마우스 명령어

좌클릭 시 빨간 점이 생기며 때기 전까지 선으로 늘어납니다.

기타

1. 슬라이스 시 정해진 모양으로 도형이 슬라이스 됩니다.
2. 도형의 날라가는 기본 속도는 0.005 입니다
3. 오각형의 경우 연속으로 3번 슬라이스 해서 사라지게 할 수 있습니다.
4. 슬라이스는 도형의 왼쪽과 오른쪽 사이 위에서 왼쪽과 오른쪽 사이 아래로 마우스를 내려야만 슬라이스 됩니다.
